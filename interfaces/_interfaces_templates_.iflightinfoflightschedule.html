Interface IFlightInfoFlightSchedule

Represents an Airline * Template flight schedule. Used to represent shape of Facebook API object. Type returned by builder and exposed to users to create own object of proper shape if desired taking into account content restrictions for the object.

Check link for content restrictions: https://developers.facebook.com/docs/messenger-platform/reference/template/airline-boarding-pass#flight_schedule
